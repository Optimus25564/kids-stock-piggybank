# 小小股票储蓄罐 (Kids Stock Piggybank)

> 面向 10 后家庭的股票储蓄产品概念原型 — Futu 产品团队内部探索

## 这是什么

一个**家长 + 孩子联名账户**形态的儿童股票储蓄产品。家长是用户,孩子参与;通过 10 年期的陪伴,在孩子 18 岁那天**自动转化为他自己的 Futu 牛牛号**。

**核心定位**:长线获客 + 家长粘性,而不是短期变现。产品本身免费,通过家长的 Futu 主账户回收 LTV。

## 当前原型包含的元素

### 首屏
- **联名账户身份条** — 实名信息("爸爸 & 小宇")常驻首屏,合规可视化
- **总资产卡** — 罐子总值 + 零花钱(可投资现金)分层显示
- **🎯 具象化目标** — "攒一台 iPhone 17 · 28%" 而不是抽象金额
- **💖 心愿单** — 多个梦想 + 孩子说的话 + 已实现的成就清单(实物 + 体验类)
- **🏠 罐子架** — 每只股票一个会动的拟物罐子,涨跌用情绪而非红绿表达
- **📖 公司卡册** — 12 张知识卡的收藏系统,读完小故事解锁新页(纯收藏驱动)
- **💬 AI 小财对话** — 强制"理由先于动作",输入框默认占位"我想买 XX,因为..."
- **🌈 成长故事集时间轴** — 从开户到 18 岁毕业的完整里程碑,最底是闪光的"专属牛牛号"目的地卡

### 入金流程(完整)
- **入金选择**(3 渠道):牛牛号调拨 / 天星银行一键入金 / FPS 转数快
- **🐮 牛牛账户调拨页** — 双账户卡 + 金额输入 + 双向余额预览 + 自动调拨开关
- **⚡ 银行一键入金页** — 已绑定提示 + 双账户卡 + 每月自动定投开关
- **🎉 入金成功页** — 庆祝动画 + 余额翻牌 + 通知小宇开关 + 去投资 CTA

### 投资中心(独立子页)
- **🪙 可投资零花钱卡** — 显示资金来源(爸爸入金 / 分红等)
- **🧒 待审批提议横幅** — 孩子的提议在等这笔钱
- **4 个决策选项** — 批准提议 / 自动加仓 / 开新罐子 / 先放着

## 设计原则

1. **整个产品没有 K 线、没有股票代码、没有红绿涨跌色** — 罐子的情绪代替金融术语
2. **孩子是主角,家长是助手** — 投资中心的默认推荐是"批准小宇的提议",不是"你自己选"
3. **每个决策都有"为什么"** — AI 强制孩子说出理由,理由本身被永久存档
4. **18 岁那天是产品的北极星** — 所有功能都为这一刻服务
